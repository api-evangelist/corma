# Corma (corma)

Corma is a SaaS license and access governance platform for modern IT teams that automates SaaS discovery, access reviews, provisioning, and deprovisioning. Recognized in the 2025 Gartner Magic Quadrant for SaaS Management Platforms, Corma helps organizations centralize all applications in one place, identify shadow IT, and reduce software spend by 10 to 15 percent.

## APIs

Corma does not currently publish a public developer API. The platform offers extensive native integrations with identity providers (Microsoft Entra ID, Okta, Google Workspace), HR platforms (Workday, BambooHR, Personio), financial tools (Spendesk, Qonto, Pennylane, Agicap), and productivity and business tools (Slack, Notion, Salesforce, HubSpot, Zendesk).
